# s1-11-m


### Para correr Front

- Ingresar a la carpeta frontend e instalar las dependencias ejecutando 
 
~~~
 npm install
~~~

-Para correr el proyecto ejecutar
 
 ~~~
 npm start
~~~
 
 ### Para correr Back

- Ingresar a la carpeta Backend e instalar las dependencias ejecutando

~~~
 npm install
~~~
- Instalar Nodemon
~~~
npm i nodemon -g
npm i nodemon
~~~
- una vez instalado ejecutar
~~~
nodemon
~~~
